# Sound Files

Para o sistema de notificações funcionar, você precisa adicionar os arquivos de som:

1. `message.mp3` - Som de mensagem recebida (curto, sutil)
2. `notification.mp3` - Som de notificação (mais chamativo)

Você pode:
- Baixar sons gratuitos de sites como freesound.org
- Usar sons do WhatsApp Web (extrair do navegador)
- Criar seus próprios sons

Enquanto os arquivos não existirem, o sistema funcionará normalmente mas sem som.
